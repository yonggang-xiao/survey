# 画图实现任务单

基于当前清洗输出，现阶段画图可以直接围绕 `output/cleaned_data.csv`、`output/multi_choice_binary.csv`、`output/q6_ranked.csv` 展开。下面是按现有文件可直接落地的画图实现任务单。

1. 任务名：单选题频率分布图（Q1、Q2、Q4、Q5）
	- 输入文件：output/cleaned_data.csv
	- 输出文件：output/desc_q1.png、output/desc_q2.png、output/desc_q4.png、output/desc_q5.png
	- 终端验收信号：打印“Q1/Q2/Q4/Q5 频数统计完成”，并分别打印每题有效样本数、选项数、图片保存路径
	- 失败时优先排查点：列名是否与主表完全一致；单选列是否含空值或前后空格；中文字体是否导致图中乱码；是否把饼图和条形图保存到了同一文件名上

2. 任务名：多选题选择次数与选择率水平条形图（Q3、Q7、Q8、Q11、Q12、Q13）
	- 输入文件：output/multi_choice_binary.csv
	- 输出文件：output/desc_q3.png、output/desc_q7.png、output/desc_q8.png、output/desc_q11.png、output/desc_q12.png、output/desc_q13.png
	- 终端验收信号：打印“Q3/Q7/Q8/Q11/Q12/Q13 多选统计完成”，并逐题打印识别到的列数、各题总选择次数、图片保存路径
	- 失败时优先排查点：是否按 Q 题号前缀正确分组列；0/1 列是否被读成字符串但未转数值；是否混入“其他”异常选项列导致标签过长或排序异常；有效问卷总数是否统一使用 160

3. 任务名：Q9 重要性维度均值雷达图
	- 输入文件：output/cleaned_data.csv
	- 输出文件：output/desc_q9_radar.png
	- 终端验收信号：打印“Q9 雷达图完成”，并输出 10 个维度的均值表、维度数量、图片保存路径
	- 失败时优先排查点：10 个 Q9 列是否全部读到；数值列是否存在空值或非数值字符串；雷达图维度顺序是否固定；中文长标签是否重叠

4. 任务名：Q9 重要性维度分布箱线图
	- 输入文件：output/cleaned_data.csv
	- 输出文件：output/desc_q9_boxplot.png
	- 终端验收信号：打印“Q9 箱线图完成”，并输出参与作图的维度数、每维非空样本数、图片保存路径
	- 失败时优先排查点：宽表转长表时是否保留了所有维度；是否误把非 Q9 数值列带入；纵轴范围是否固定在 1 到 5；标签旋转后是否仍不可读

5. 任务名：Q10 多主体平均权重堆积条形图
	- 输入文件：output/cleaned_data.csv
	- 输出文件：output/desc_q10.png
	- 终端验收信号：打印“Q10 权重图完成”，并输出 5 个主体平均权重、均值合计、图片保存路径
	- 失败时优先排查点：5 个 Q10 列是否完整；原始权重是否已是数值型；均值合计是否接近 100；若明显偏离，先回查清洗阶段有没有异常记录未处理

6. 任务名：Q6 排序题加权偏好得分柱状图
	- 输入文件：output/q6_ranked.csv
	- 输出文件：output/desc_q6.png
	- 终端验收信号：打印“Q6 加权得分完成”，并输出识别到的成果类型数量、每类总得分、图片保存路径
	- 失败时优先排查点：第1选/第2选/第3选三列是否都有值；权重是否严格按 3/2/1；同一成果类型文本是否存在细微差异导致被拆成多个类别；排序是否按总分降序

7. 任务名：图表统计汇总表生成
	- 输入文件：output/cleaned_data.csv、output/multi_choice_binary.csv、output/q6_ranked.csv
	- 输出文件：output/descriptive_summary.csv
	- 终端验收信号：打印“descriptive_summary.csv 已保存”，并输出汇总表行数、包含的题号清单、保存路径
	- 失败时优先排查点：不同题型统计结果字段是否统一；题号命名是否一致；单选、多选、量表、排序题的统计口径是否混淆；是否遗漏 Q9 和 Q10 这种非频数型结果
